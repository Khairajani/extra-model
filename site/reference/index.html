cluster_adjectives(adjective_counts, vectorizer)

Cluster adjectives based on a constant radius clustering algorithm.

Technical implementation uses a scikitlearn BallTree.

Parameters:

Name Type Description Default
adjective_counts [(str,int)]

dictionary with adjectives and their counts

 required
vectorizer Union[:class:`_vect,izer.vect,izer`]

provide embeddings to evaluate adjective similarity

 required

Returns:

Type Description
([string],[int],[[(string,int)]])

first list contains the representative words for the adjective clusters, second list the size (total number of clustered adjectives) for each cluster, third list has for each cluster a list of the constituent adjectives with number of occurence

fill_sentiment_dict(adjective_counts)

Given a dictionary with adjectives and their counts, will compute.

The sentiment of each of the adjectives using the VADER sentiment analysis package and return a dictionary of the adjectives and their sentiments.

Parameters:

Name Type Description Default
adjective_counts dict

dictionary with adjectives and their counts

 required

Returns:

Type Description
dict

dictionary, where the keys are the adjectives and the values are tuples of the corresponding compound sentiument and binary sentiment

adjective_info(dataframe_topics, dataframe_aspects, vectorizer)

Add adjective related information to the dataframes.

This has two facets: -> for each topic cluster similar adjectives, to get a more abstract/readable list -> for each topic, use the adjectives to come up with a sentiment classification
